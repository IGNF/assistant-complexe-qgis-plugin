
# IGN_assistant-complexe

This plugin facilitates the management of the relationship between road segments and numbered or named roads,
and allows users to add or remove one or more segments from these numbered or named roads,
as well as select all the segments that make them up.

## Pré-requis

Mandatory : The "plugin\_maitre" must be installed.
Link to the plugin maître : [maitre-qgis-plugin sur GitHub](https://github.com/IGNF/maitre-qgis-plugin)

## Fonctionnalités

French : 
* Ajouter un complexe à une route numérotée ou nommée
* Retirer un complexe d'une route numérotée ou nommée
* Affichage du chemin le plus court entre 2 tronçons de route


English :
* Add a complex to a numbered or named road.
* Remove a complex from a numbered or named road.
* Display of the shortest path between two road segments.
